# HW3
# Генератор и аналитик данных о музыке

Генерирует CSV с треками (название, исполнитель, продолжительность, жанр) и создаёт HTML-отчёт.

## Команды
```bash
./run.sh build_generator   # собрать генератор
./run.sh run_generator     # создать data.csv
./run.sh build_reporter    # собрать аналитика
./run.sh run_reporter      # создать report.html
./run.sh clear_data        # очистить data/
./run.sh structure         # показать структуру
./run.sh inside_generator  # войти в контейнер генератора
./run.sh inside_reporter   # войти в контейнер аналитика
```
